# Discrete function

## Problem statement

Find all of the inversions in a given array.

**Input**

There is an N as the argument, that is an array size. Than N lines as an input
with the values of the array elements.

**Output**

The number of inversion in a given array.

## Sample

**Input**

    $ ... 3
    1
    3
    2

**Output**

    1
